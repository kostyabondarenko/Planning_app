# Название проекта: Planning

Это проект для планирования и изучения работы LLM агентов.

## Как запустить
1. **Backend**:
   - Перейдите в папку `backend`.
   - Установите зависимости: `pip install -r requirements.txt`.
   - Запустите сервер: `uvicorn app.main:app --reload`.
2. **Frontend**:
   - Перейдите в папку `frontend`.
   - Установите зависимости: `npm install`.
   - Запустите: `npm run dev`.
3. **База данных**:
   - Используйте `docker-compose up -d` в корне проекта.


## Для AI-агентов
Если ты — AI-агент, **ОБЯЗАТЕЛЬНО** начни с прочтения файла `agents.md` в корне проекта. Там написано, как ты должен работать.
