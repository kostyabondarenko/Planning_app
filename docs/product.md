# Навигатор Целей (Goal Navigator)

## Суть продукта

Приложение для движения к большим целям через понятные вехи и регулярные действия. Связывает повседневную активность с общей картиной, делая путь прозрачным и мотивирующим.

---

## Основные страницы

### 1. Цели
Управление целями, их вехами и регулярными действиями.
- Цели с названием и сроками
- Вехи (milestones) для каждой цели
- Регулярные действия для вех с привязкой к дням недели
- Индивидуальный целевой процент (target_percent) для каждого действия
- Визуализация прогресса с маркером целевого процента

*Подробнее: `plans/002-goals-page.md`*

### 2. Календарь
Обзор планов в формате календарной сетки.
- Месячная сетка
- Отображение целей и вех в рамках выбранного месяца

*Подробнее: `plans/calendar-page.md`*

### 3. Ближайшие дни
Оперативное планирование в формате канбан-доски.
- Действия на ближайшую неделю
- Быстрое управление задачами

*Подробнее: `plans/upcoming-page.md`*
